Scroll Nav

Scroll nav plugin provides scrolling to the targeted block with smooth animation.

How to use?

Wrap any block in a parent element with an ID or class and add the same ID or the class in the JS init function of the assets/vendor/hs-scroll-nav/dist/hs-scroll-nav.min.js plugin, add #section-name to the href element you want to be scrolled to a certain content.

Copy-paste the following <script> near the end of your pages under JS Implementing Plugins to enable it.

              
                <script src="../../assets/vendor/hs-scroll-nav/dist/hs-scroll-nav.min.js"></script>

Copy-paste the following <script> near the end of your pages under JS Front to enable it.

              
                <script>
                  $(document).on('ready', function () {
                    // initialization of scroll nav
                    $('.js-scroll-nav').each(function () {
                      var scrollNav = new HSScrollNav($(this)).init();
                    });
                  });
                </script>

Basic example

Scroll the area below the navbar or click on the menu to watch the smooth scrolling in action. It can also be used in the dropdowns.

JavaScript behavior

Extend

Plugin methods are called using a wrapper:

              
                <script>
                  $(document).on('ready', function () {
                    // initialization of scroll nav
                    $('.js-scroll-nav').each(function () {
                      var scrollNav = new HSScrollNav($(this)).init();
                    });
                  });
                </script>

Attributes

By assigning a variable, you can call the standard methods of the plugin:

              
                data-hs-scroll-nav-options='{
                 ...
              }' - array

Methods

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-easing="".

jquery.easing.min.js plugin is required to extend this plugin.

JS parameters Description Default value

duration

 Speed of the animation 400

easing

 Slowdown function for transition 'linear'

sectionClass

 Class of elements to which navigation will be scrolled '.scroll-nav-section'

customOffsetTop

 When scrolling, this value will be added to the final viewport offset value 0

activeItemClass

 Class added to the active navigation element 'active'

beforeShow: function (invoker) {}

 Class added to the current section 'active'

afterShow

 Function performed after the viewport moves to the desired location function() {}

beforeShow

 Function performed before viewport shifts function() {return true;}

beforeShow: function (invoker) {}

 Element whose height will be taken into account when shifting the viewport $('.header')
